# 你好，我是三岁

**中文** · [English](README.md)

从 HR 转向 AI 产品，所有产品都是 AI 写的，我只负责想。

对我来说，从 HR 转向 AI 产品，用 vibe coding 的方式完成产品思考，把清晰判断、实际需求和快速迭代变成真正可用的产品。

我不是传统意义上的工程师，也不把自己定义成手写代码的开发者。我发现真实世界的问题，有了想法，找到场景，测试结果，然后交给 AI 编程模型，AI 就是我的实现伙伴。

## 我在做什么

我关注轻量、本地优先、服务于日常文档、文件、演示和知识工作流的小工具：

- 文件转换和批处理工具
- Markdown、Obsidian 和文本工作流
- CSV、JSON、压缩包和文件名清理
- 多语言桌面端或浏览器工具
- 给非工程师使用的实用软件

## 我的工作方式

- 从真实工作流里的摩擦出发，而不是从抽象功能清单出发
- 把模糊想法整理成清晰的产品提示词、用户流程和验收标准
- 使用 AI 编程模型完成构建、调试、打包和改进
- 在真实场景里试用工具，判断它是否真的好用
- 保持工具足够简单，让只想完成任务的人也能直接上手

## 当前兴趣

Vibe coding、AI 辅助产品构建、本地优先工具、Markdown 工作流、文件转换、数据清洗、多语言界面，以及帮助非工程师完成实际工作的软件。
